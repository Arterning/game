# 坦克大战 (Tank Battle)

一个使用 Python 和 Pygame 开发的经典坦克对战游戏。

## 游戏特性

- 玩家控制的坦克与 AI 敌人对战
- 智能敌人 AI：随机移动（每2秒换方向）和随机射击
- 多种地图元素：
  - 可破坏的墙壁（砖块）
  - 不可破坏的墙壁（石头）
  - 草丛（可以隐藏坦克，但不阻挡移动）
- 关卡系统：
  - 击败所有敌人进入下一关
  - 每关敌人数量增加
  - 每关敌人移动速度提升
- 生命系统：
  - 玩家有3条生命
  - 每条生命有3点血量
  - 血量耗尽会消耗一条生命并重置位置
- 游戏结束界面，可以重新开始或退出

## 安装依赖

确保你已经安装了 Python 3.13 或更高版本。

使用 pip 安装依赖：

```bash
pip install pygame
```

或者使用 uv（如果你在使用 uv 管理项目）：

```bash
uv pip install -e .
```

## 如何运行

```bash
python main.py
```

## 游戏操作

### 移动控制
- **W / ↑**：向上移动
- **S / ↓**：向下移动
- **A / ←**：向左移动
- **D / →**：向右移动

### 战斗控制
- **空格键**：发射子弹

### 游戏状态控制
- **游戏结束时**：
  - **R**：重新开始游戏
  - **Q**：退出游戏
- **关卡完成时**：
  - **空格键**：进入下一关

## 游戏元素说明

### 坦克
- **绿色坦克**：玩家控制的坦克
- **红色坦克**：敌人坦克（AI 控制）

### 子弹
- **黄色子弹**：玩家发射的子弹
- **红色子弹**：敌人发射的子弹

### 墙壁
- **棕色砖块**：可以被子弹摧毁
- **灰色石头**：不可摧毁，阻挡坦克和子弹
- **深绿色草丛**：不阻挡移动，但可以遮挡坦克（视觉效果）

## 游戏规则

1. 消灭所有敌人坦克即可进入下一关
2. 每关敌人数量会增加
3. 每关敌人移动速度会提升
4. 玩家有3条生命，每条生命3点血量
5. 被敌人子弹击中会损失血量
6. 血量耗尽会损失一条生命并重置位置
7. 所有生命耗尽则游戏结束

## 开发说明

游戏采用面向对象设计：
- `Tank`：坦克基类
- `PlayerTank`：玩家坦克类
- `EnemyTank`：敌人坦克类（包含 AI 逻辑）
- `Bullet`：子弹类
- `Wall`：墙壁类
- `Game`：游戏主类，管理游戏状态和循环
